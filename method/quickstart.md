# 快速入门

## 准备工作环境

有两种方案，可以将此 API 项目保存到本地并直接运行, 或者你可以直接使用我的链接，这可以为你节省时间

## 本地使用

只需下载文件并运行：`python main.py`


> 注意事项
> 
> 运行前需要安装一些用到的库，本项目使用`fastapi`，可以运行试试看是否还有其它错误，有的话请自行处理，或者问问我？
> 
如果发现程序还是运行不了，可以阅读我的程序的注释，例如，您可能需要要求更改 Pixiv 的代理地址或 JSON 文件。
